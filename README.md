   ``` terminal 
   >git clone https://github.com/BOLT4L/solar-challenge-week1/
   > cd solar-challenge-week1
2. Set up a virtual environment and activate :
   ``` terminal 
   > python -m venv .venv
   > .venv\scripts\activate 
3. Install dependencies :
   ``` terminal
   > pip install -r requirements.txt
